Прогнозирование раннего выхода на пенсию

Добро пожаловать в API для прогнозирования раннего выхода на пенсию! Используйте этот API для загрузки данных и получения прогноза.

Endpoints

POST /predict-dataset/

Принимает два файла с датасетом (CSV) и возвращает прогнозы для каждого счета.

Пример использования: Загрузите CSV-файл, содержащий данные о нескольких людях, чтобы получить прогнозы для каждого.

Пример ответа:

[ {"id": 1, "prediction": "значение прогноза"}, {"id": 2, "prediction": "значение прогноза"} ]

Инструкции по использованию

Для начала работы отправьте POST-запрос на один из эндпоинтов, указанных выше. Убедитесь, что данные форматированы правильно.

  • Для /api/predict-dataset/: отправьте файл в формате CSV, содержащий необходимые данные.
